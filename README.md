# pygame-snake-2-player
2 player snake game

py -m pip install -U pygame --user

2 player Snake Game: (By Ahunjan)
About the Game:

Game controls:
  Player 1: ← ↓ ↑ →   keys
  Player 2: ‘a’ ‘s’ ‘w’ ‘d’ keys

Snake can die in any of the following scenarios:
  Hits the wall / boundary
  Hits itself
  Hits the opponent Snake
